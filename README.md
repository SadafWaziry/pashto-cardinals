Pashto Cardinals
=================
A package that can convert a number to Pashto language


Install
=========
Via Pip:

`$ pip install pashtocardinals`

Usage
======
``` python
from pashtocardinals import *

pashtocardinals.convert(234)
```

The above code will return "دوه سوه او څلور دیرش"
