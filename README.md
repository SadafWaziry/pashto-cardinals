Pashto Cardinals
=================
A package that can convert a number to Pashto language


Install
=========
Via Pip:

`$ pip install pashto_cardinals`

Usage
======
``` python
from pashto_cardinals import *

pashto_cardinals.convert(234)
```

The above code will return "دوه سوه او څلور دیرش"
